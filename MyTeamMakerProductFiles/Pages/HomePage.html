What is this Website?

This Website is a Flexible Platform which can be used to form teams based on the abilities of each individual. Furthermore, this website's unique attribute-based approach allows users to create the guidelines for the formation of any team in any situation! Thus, this website enables the users to make teams, anytime, and anywhere and for any occasion.

Using the Website

The website is very userfriendly. The "Make Your Teams" page contains a large body of text that you can read and use. All the places where you need to provide input are clearly displayed. Thus, you don't need to worry about missing out on giving any important information! However, if you wish for more information, please refer to the Help Page using the Navigation bar! You will find more detailed steps on how to operate this program.

The Inspiration

At the start of EngHack, I did not know anybody. Despite Networking, I was unable to find a team which shared the same interests as I did and also worked towards a product I could contribute towards. Therefore, I saw a need for a product that can help match people on the basis of both, their interests, as well as their skills. Hence, MyTeamMaker was born!
